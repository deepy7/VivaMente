# VivaMente

VivaMente es una aplicación web desarrollada como Trabajo de Fin de Grado en Ingeniería Informática. Su objetivo es ofrecer una herramienta sencilla, visual y accesible de entrenamiento cognitivo para personas con Alzheimer en fases tempranas y moderadas.

La aplicación permite realizar ejercicios cognitivos básicos, registrar resultados y consultar la evolución de los usuarios, facilitando también el seguimiento por parte de familiares o cuidadores.

## Aplicación publicada

La aplicación está desplegada en Netlify y puede consultarse en el siguiente enlace:

https://vivamente-tfg.netlify.app/login

## Tecnologías utilizadas

- React
- TypeScript
- Vite
- Tailwind CSS
- Supabase
- Netlify

## Funcionalidades principales

- Registro e inicio de sesión de usuarios.
- Diferenciación entre rol de usuario y rol de cuidador.
- Realización de ejercicios cognitivos.
- Registro de resultados.
- Consulta de evolución del usuario.
- Gestión de usuarios asignados para cuidadores.
- Sección de ayuda y soporte.

## Ejercicios cognitivos incluidos

La aplicación incluye ejercicios orientados a trabajar distintas áreas cognitivas:

- Memoria visual.
- Asociación imagen-palabra.
- Atención selectiva.
- Emparejamiento de objetos.
- Reconocimiento de imágenes.
- Secuencia visual.

## Instalación y ejecución en local

Para ejecutar el proyecto en local, es necesario tener instalado Node.js.

1. Clonar el repositorio:

```bash
git clone https://github.com/deepy7/VivaMente.git
```

2. Acceder a la carpeta del proyecto:

```bash
cd VivaMente
```

3. Instalar las dependencias:

```bash
npm install
```

4. Ejecutar el proyecto en modo desarrollo:

```bash
npm run dev
```

5. Generar la versión de producción:

```bash
npm run build
```

## Configuración

El proyecto utiliza Supabase como backend y base de datos. Para ejecutar la aplicación en local correctamente, es necesario configurar las variables de entorno correspondientes en un archivo `.env`.
